# BitFlowLM 🧠💻

**An Agentic, NotebookLM-Inspired Visualizer and Chatbot for Computer Science**

BitFlowLM is a powerful, zero-setup, single-file web application designed to help Computer Science students and professionals understand complex topics. Upload your PDFs, chat with an AI agent powered by Gemini 2.5 Flash, and watch as it dynamically generates real-time, animated canvas visualizations of Data Structures, Algorithms, Networks and OS concepts
